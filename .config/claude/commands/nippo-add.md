# 日報に追記する

現在の日報ファイル（ ~/Obsidian/02_Daily/nippo.$(date +%Y-%m-%d).md）に以下の内容を追記してください。

## 追記する内容: $ARGUMENTS

まず、日報ファイルが存在するか確認し、存在しない場合は新規作成してください。

### 既存ファイルへの追記の場合:

追記内容の先頭キーワードによって自動振り分け：

**時系列ログ（デフォルト）:**

1. 「## 作業ログ（分報・思考メモ）」セクションを探す
2. 最後に以下を追記:

```markdown
- $(date +%H:%M) $ARGUMENTS
```

**特別な処理:**

- `$ARGUMENTS` が「タスク:」で始まる → 今日のタスクに追記
- `$ARGUMENTS` が「フォーカス:」で始まる → 今日のフォーカスに追記
- `$ARGUMENTS` が「成果:」で始まる → 成果とアウトカムに追記
- `$ARGUMENTS` が「学び:」で始まる → 学び・気づきに追記
- `$ARGUMENTS` が「評価:」で始まる → 自己評価を更新
- `$ARGUMENTS` が「ギャップ:」で始まる → ギャップと次の一手を更新
- `$ARGUMENTS` が「明日:」で始まる → 明日へのフォーカスを更新
