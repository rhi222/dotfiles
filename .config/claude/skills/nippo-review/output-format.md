# 評価面談材料 出力フォーマット

標準出力に以下のフォーマットで表示してください（日報には追記しない）：

```markdown
## 📝 評価面談材料 - 過去{n}日間

**分析期間**: YYYY-MM-DD 〜 YYYY-MM-DD
**分析対象**: {n}件の日報

---

### 🏆 主要成果リスト

> 期間中に完了した主要な成果。評価面談で具体例として使用できます

1. **[成果名]** (YYYY-MM-DD)
   - 内容: [具体的な内容]
   - インパクト: [影響範囲]

2. **[成果名]** (YYYY-MM-DD)
   - 内容: [具体的な内容]
   - インパクト: [影響範囲]

---

### 📈 成長した領域

> 期間を通じて成長が見られる領域。具体的なエピソードで裏付け

- **[領域名]**: [前半の状態] → [後半の状態]
  - 根拠: [日報からの具体的エピソード]

---

### 🔢 数値化できるインパクト

> 定量的に示せる成果・貢献

| 項目 | 数値 | 備考 |
| ---- | ---- | ---- |
| [項目名] | {n} | [補足] |

---

### 🚀 挑戦した領域

> 新しい取り組みや困難への挑戦

- **[挑戦名]** (YYYY-MM-DD頃)
  - 内容: [何に挑戦したか]
  - 結果: [どうなったか]
  - 学び: [何を得たか]

---

### ⚡ 自己評価に使えるキーワード

> 評価文を書く際のヒント（評価文そのものではありません）

- [キーワード1]: [関連する成果・エピソード]
- [キーワード2]: [関連する成果・エピソード]
```
