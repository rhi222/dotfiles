# Daily Growth Log テンプレート

新規日報ファイル作成時に使用するテンプレート:

```markdown
# Daily Growth Log

Date: $(date +%Y-%m-%d)

## 今日のフォーカス（1行）

> nippo-goals.md の4軸から今日意識する軸を選ぶ
> 例: 「横断: サブシステム構成図のレビュー」

-

## 今日の予定

-

## 今日のおすすめタスク（目標逆算）

> nippo-goals.md の達成基準・行動習慣から自動提案。やらなくてもOK、意識するだけでも価値がある

- [ ] {AI提案タスク1 - 達成基準由来}
- [ ] {AI提案タスク2 - 行動習慣由来}

## 今日のタスク

> 今日「やること」と「期待するアウトカム」
> 凡例: [x]: 完了, [ ]: 未完了, [-]: 部分的に完了

前日からの引き継ぎ

> 前営業日の未完了タスク・「次の一手」から自動転記。該当なしの場合はこのセクションを省略する

- [ ] {未完了タスク} ← M/D引き継ぎ
  - 期待アウトカム：{元の期待アウトカム}
- [ ] {次の一手アクション}（{見積もり時間}） ← 次の一手
  - 期待アウトカム：{Why}

自分でやるタスク

- [ ] タスク①
  - 期待アウトカム：

claudeに任せるタスク

- [ ] タスク
  - 期待アウトカム：

codexに任せるタスク

- [ ] タスク
  - 期待アウトカム：

参考: notionカンバンボード: https://www.notion.so/forcia-1dca17662dc7807b99bdc73aaa42d3c0?p=2e2a17662dc780e28825de9d4bfcfdb4&pm=s

## 作業ログ（分報・思考メモ）

> その時考えたこと・気づいたことを時系列で。きれいに書かなくてOK

- $(date +%H:%M) $ARGUMENTS

## 作業メモ

> 作業時の自由記述欄。長くなるなら別記事推奨
```
