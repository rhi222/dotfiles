# 今日の予定をGoogle Calendarから生成する

新規日報ファイル作成時に、`## 今日の予定` セクションへ埋め込む内容を Google Calendar から生成するためのロジック。

## 取得

`mcp__claude_ai_Google_Calendar__list_events` で当日（`$(date +%Y-%m-%d)`）の予定を取得する。

- `startTime`: `YYYY-MM-DDT00:00:00`
- `endTime`: `YYYY-MM-DDT23:59:59`
- `timeZone`: `Asia/Tokyo`
- `orderBy`: `startTime`
- `pageSize`: `100`

未認証等で取得できなかった場合は、`## 今日の予定` セクションを以下の1行のみにして**日報作成自体は続行する**（中断しない）:

```markdown
- 取得失敗: Google Calendarに接続できません。
```

## 一覧に載せる予定（会議・MTG）の条件

- 自分（`self: true`）が参加予定の会議/MTGのみ
- 次のキーワードを含む予定は一覧から除外: 外出 / 移動 / 私用 / 離席 / ブロック / 作業時間 / 集中 / 勉強 / 自習 / 不在 / `[個人]` / `[作業]` / `[ブロック]`
- 終日予定（`date` のみ）/ キャンセル（`status: cancelled`）/ 自分が辞退（`responseStatus: declined`）は除外
- タイトルに「スキップ」等で当日中止が明記された予定（例: `【今日はスキップ】…`）は除外
- `eventType` が `OUT_OF_OFFICE` / `FOCUS_TIME` / `WORKING_LOCATION` 等の私用・ブロック系は一覧から除外（ただし埋まり扱いには含める。下記参照）

## 作業可能時間の計算

- 営業時間: **09:30〜19:00**
- 「埋まり」として扱う予定:
  - 一覧に載る会議/MTG
  - 一覧からは除外したが時間を占有する予定（OUT_OF_OFFICE、私用、除外キーワードを含む予定 など）
  - ただし **キャンセル / 自分が辞退 / 当日スキップ明記** の予定は占有に含めない（その枠は空き扱い）
- 埋まり予定を開始時刻でソートし、営業時間（09:30〜19:00）にクリップしたうえで、重複・連続する枠をマージする
- マージ後、以下の空き時間を抽出する:
  - 営業開始〜最初の予定
  - 予定と予定の間
  - 最後の予定〜営業終了
- **45分以上**の空き時間だけを「ブロック可能時間」とする

## 合計時間

- `mtg時間`: 一覧に載る会議/MTG の時間枠をマージ（重複は二重計上しない）した合計
- `作業可能時間`: ブロック可能時間（45分以上の空き）の合計

## `## 今日の予定` への埋め込みフォーマット

時刻は24時間表記・2桁ゼロパディング。予定が無い / ブロック可能時間が無い場合は該当行に `なし` と記載する。

```markdown
## 今日の予定

### 会議・MTG

- hh:mm ~ hh:mm: 予定タイトル

### ブロック可能時間

- hh:mm ~ hh:mm: 作業ブロック可能(N分)

### 合計時間

- mtg時間: X h Y m
- 作業可能時間: N h M m
```
